# Controle Financeiro

## Descrição
Sistema de controle de entradas e saídas financeiras com cálculo automático de saldo.

## Funcionalidades
- Registro de receitas e despesas
- Cálculo de saldo
- Organização financeira básica

## Tecnologias
- Excel
  Arquivo xlsx

## Objetivo
Projeto voltado para gestão financeira e estruturação de dados.
